## Overview
This repository documents the technical design for a community-focused
application intended to provide bin collection reminders and waste-sorting
guidance for residents in Salford.

The project aims to improve waste collection efficiency, reduce missed
collections, and increase recycling awareness.

## My Role
I led the technical design of the application, including system architecture,
data modelling, and service-level logic suitable for local authority use.

## Civic Impact
The project is designed to support residents and local councils by reducing
service friction and improving communication around waste management.

## Project Status
This project is currently in the design and pre-implementation phase.
